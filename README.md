# PRODIGY_GAI_01
Fine-tuned the GPT-2 model on a custom dataset to generate coherent and contextually relevant text that mimics the style and structure of the training data.

In this project, I used the GPT-2 model to generate text from user prompts. I fine-tuned the model on a custom dataset and learned how AI can generate meaningful and contextually relevant text. This task helped me understand the basics of text generation and transformer models.

Tools Used
Python
Google Colab
GPT-2
